# 🚀 AI Interview Preparation Platform

A full-stack **GenAI-powered web application** that helps users prepare for job interviews by analyzing resumes and job descriptions to generate personalized interview reports, skill gap insights, and preparation roadmaps.

---

## 🌟 Features

- 📄 Resume Analysis – Upload PDF resumes and extract key information  
- 🎯 Job Description Matching – Compare resume with job role requirements  
- 📊 Match Score & Skill Gap Detection – Identify strengths and missing skills  
- 🤖 AI Interview Questions – Generate technical & behavioral questions using GenAI  
- 🗺️ Preparation Roadmap – Get a structured plan for interview preparation  
- 📥 PDF Export – Download AI-generated reports as professional PDFs  
- 🔐 Secure Authentication – JWT-based login & session management  

---

## 🏗️ Tech Stack

**Frontend:**  
React.js, Vite, SCSS, Axios  

**Backend:**  
Node.js, Express.js  

**Database:**  
MongoDB  

**Authentication:**  
JWT (jsonwebtoken), bcrypt  

**AI Integration:**  
Google Gemini API  

**File Handling & Processing:**  
Multer, pdf-parse  

**PDF Generation:**  
Puppeteer
